## 💻 Projeto

O Let Me Ask é um projeto que foi desenvolvido durante a Next Level Week #6, apresentado pela  Rocketseat durante os dias 20 a 27 de Jumho de 2021.

```

# ou
$ yarn install

# Executando a aplicação
$ npm start
# ou
$ yarn start
```
